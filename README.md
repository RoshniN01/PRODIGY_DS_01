# PRODIGY_DS_01 - Data Visualization using Spotify Dataset

## Objective

The objective of this task was to perform Exploratory Data Analysis (EDA) and create meaningful visualizations using the Spotify Tracks dataset. The analysis focuses on understanding song popularity, genre trends, artist distribution, and explicit content through graphical representations.

---

## Dataset

- **Dataset:** Spotify Tracks Dataset
- **Records:** 6,300 
- **Features:** 8 

---

## Tools Used

- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

---

## Data Preprocessing

- Loaded the dataset using Pandas.
- Checked the dataset structure and data types.
- Verified missing values.
- Verified that the dataset contains no duplicate records.
- Prepared the dataset for visualization.

---

## Visualizations

### 1. Distribution of Song Popularity
- Created a histogram with KDE to analyze the distribution of song popularity.

### 2. Top 10 Genres by Average Song Popularity
- Compared the average popularity scores across genres using a horizontal bar chart.

### 3. Top 10 Artists by Number of Songs
- Identified artists with the highest number of songs in the dataset.

### 4. Explicit vs Non-Explicit Songs
- Compared the number of explicit and non-explicit songs using a count plot.

---

## Key Insights

- Most songs have low to moderate popularity, while only a few songs achieve very high popularity.
- Rock, Dance, and Pop genres show the highest average popularity among the top genres.
- A small number of artists contribute a large number of songs to the dataset.
- The dataset contains significantly more non-explicit songs than explicit songs.

---

## Conclusion

This analysis provided valuable insights into Spotify music trends through exploratory data analysis and visualization. The visualizations revealed patterns in song popularity, genre performance, artist contributions, and explicit content distribution. These insights help in understanding the characteristics of the dataset and form a strong foundation for future predictive analytics and recommendation system projects.
